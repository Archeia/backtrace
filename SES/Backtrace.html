Backtrace

Provides methods and configuration options for the full error backtrace.

Constant Summary

RAISE_EXCEPTIONS =

Whether or not to explicitly raise exceptions after exception handling has taken place; by default, this is set to true.

This script will attempt to work around any exceptions raised if this constant is set to a false value; this may be useful for script authors or game developers for debugging purposes, but may cause game instability (or other odd behaviors).

It is highly recommended that you set this constant to false before releasing any version of your project.

true
LOG_EXCEPTIONS =

Whether or not to log exception information and the backtrace to a log file; may be either true or false.

false
APPEND_LOG =

Whether to append to the log file or overwrite previous contents whenever an exception is handled; set to true to append to the log file, false to overwrite its contents. The value of this constant only applies if LOG_EXCEPTIONS has been set to a true value.

true
LOG_FILE =

The log file exception information is written to if the LOG_EXCEPTIONS constant is set to a true value. The placement of this file is relative to your project's root directory.

'Backtrace.log'
ALERT =

Creates an alert box containing information about a caught exception when one is handled if RAISE_EXCEPTIONS is set to a false value. This may be useful to alert developers or play testers at the exact moment an exception occurs.

false

def self.print_caught(exception, backtrace, stream = STDERR)
  msg =  Time.now.to_s << ' >> EXCEPTION CAUGHT <<'
  msg << "\n#{exception.class}: #{exception}.\nBacktrace:\n\t"
  msg << (backtrace.join("\n\t"))
  stream.puts msg
end

def self.with_exception_handling
  yield
rescue RGSSReset
  # NOTE: This block assumes that `rgss_main` has the default block given
  # to it by the editor -- this is generally a reasonable assumption.
  Graphics.transition(10)
  SceneManager.run
rescue SystemExit
  exit
rescue Exception => ex
  trace = []
  ex.backtrace.each do |line|
    break if line[/^:1:/] # Information past this point is irrelevant.
    trace << line.gsub(/^{(\d+)}/) { $RGSS_SCRIPTS[$1.to_i][1] }
  end
  print_caught(ex, trace)
  File.open(LOG_FILE, APPEND_LOG ? 'a' : 'w') do |file|
    print_caught(ex, trace, file)
  end if LOG_EXCEPTIONS
  RAISE_EXCEPTIONS ? raise(ex) : (alert_caught(ex) if ALERT ; retry)
end
